DOrSU Program Recommender

Python Flask HTML & CSS JavaScript Render

Problem

Incoming college students at Davao Oriental State University often struggle to choose the right academic program. With multiple offerings and limited guidance, many rely on word-of-mouth or guesswork. The goal was to build a tool that uses data to guide students toward a program that fits their interests and strengths.

Approach

I designed a questionnaire-based system that asks students about their interests, academic preferences, and strengths. Each answer maps to weighted criteria across all DOrSU programs. A matching algorithm computes a compatibility score for each program and presents the top recommendations.

Key design decisions:

  • Question clarity — questions were iterated to be clear, unbiased, and easy to answer
  • Weighted scoring — different criteria carry different importance depending on the program
  • Simple UI — a distraction-free interface that keeps the student focused on the questionnaire

Implementation

Built with Flask, the app handles form submissions, computes scores server-side, and renders results through Jinja2 templates. The front-end uses vanilla JavaScript for interactivity with a responsive layout. Deployment is on Render's free tier.

Result

The system provides personalized program recommendations in real-time. It gives incoming students a data-informed starting point for one of the most important decisions in their academic journey.
